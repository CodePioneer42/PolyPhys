## Problem Framing
- Phenomenon of Interest: Cohesin 敲除后，Hi-C 中 TAD/loop 明显减弱或消失，同时 A/B compartment 分离信号增强。
- Scope and scale: 间期哺乳动物细胞核尺度；先在单染色体或多染色体 coarse-grained 聚合物层面复现接触图谱统计特征，不追求逐位点精确拟合。
- Known constraints:
  - Cohesin 主要与 loop extrusion 相关，通常需要 ATP 驱动装载/转运过程。
  - compartment 主要由染色质状态差异（A/B 类型）与相互作用偏好产生。
  - TAD 与 compartment 可由不同机制主导，存在可分离性。
- Neutral restatement: 目标不是“证明唯一机制”，而是构建最小可证伪模型，解释为何移除 Cohesin 会削弱局部挤出结构（TAD）并同时让远程区室分离更显著。
- Missing assumptions to clarify:
  - 细胞系统已锁定为 HCT116 急性降解（Acute depletion）
  - 是否有 CTCF ChIP-seq、ATAC/H3K27ac 等轨道可用于参数化 A/B 类型
  - 关注稳态群体 Hi-C 还是时间序列重建
- Chosen path: physical mechanism polymer model（并预留 data-driven 参数化接口）
- System lock-in note:
  - 推荐对应实验设定为 HCT116 RAD21/SCC1 急性降解时间窗（例如 0h 对照 vs 6h 处理）以减少细胞周期重分布干扰。

## Competing Mechanisms
- Candidate A: 挤出移除 + 区室“去屏蔽”机制
  - WT 中 Cohesin loop extrusion 形成局部结构（TAD），并在一定程度上混合/跨越不同 compartment，部分“遮蔽”长程 A-A/B-B 偏好。
  - KO 后挤出消失，TAD 边界与局部增强点丢失；链段更充分按 A/B 相互作用重排，compartment 信号增强。
- Candidate B: 挤出移除 + 有效温度/流动性重分配机制
  - Cohesin 相关活性过程在 WT 引入额外 nonequilibrium 扰动，使区室分离受抑。
  - KO 后系统更接近平衡驱动的微相分离，A/B 对比增强。
- Degeneracy warning: 两个候选机制都可解释“TAD 下降 + compartment 上升”，仅靠单一稳态 Hi-C 难以区分；必须加入正交观测和扰动实验。

## The Minimal Physical Model
- Base Polymer Model:
  - 染色体表示为 block-copolymer bead-spring 链。
  - 每个 bead 带 A/B 类型标签（可先用公开 compartment 注释或简单二值块状序列）。
  - 局部连接：FENE/谐弹簧 + 排斥体积；可选弯曲刚度设定持久长度。
- Baseline Control:
  - 去除 Cohesin/挤出模块，仅保留非特异排斥或极弱统一吸引（不区分 A/B）。
  - 该控制用于确认“无机制时既无 TAD 也无显著 compartment”的基线。
- Experimental Perturbation:
  - WT 条件：加入 loop extrusion（挤出子密度、步进率、驻留时间、CTCF 屏障方向性）。
  - KO 条件：将挤出率或挤出子密度降为 0，同时保留 A/B 互作偏好（epsilon_AA < epsilon_AB < epsilon_BB 或对称 A-A/B-B 增强）。
  - 可选第二扰动：在 KO 上再降低 A/B 选择性，验证 compartment 增强是否被抹平。
- Active Components Required:
  - 对 Candidate A：需要（WT 有 ATP 驱动挤出；KO 关闭该活性模块）。
  - 对 Candidate B：强需要（需显式比较 active noise/nonequilibrium flux 在 WT 与 KO 的差异）。
- Active Module Checklist:
  - loop extrusion: 是（核心模块）
  - nonequilibrium fluxes: 建议在扩展模型中显式测试（区分 A vs B）
  - reader-writer feedback: 初版否（避免过拟合）；二期可加入表观状态传播
  - viscoelastic/environmental couplings: 初版可忽略；若时间尺度失配再加入

## Observables & Validation Strategy
- Fitting Observables:
  - Hi-C 距离衰减曲线 P(s)
  - insulation score / directionality index 的尺度分布
  - compartment eigenvector（PC1）及 saddle plot 强度
- Orthogonal Validation:
  - DNA-FISH/oligoFISH：A-A 与 B-B 空间距离分布
  - 单细胞 Hi-C 中 compartment strength 分布宽度
  - Lamin 相关定位（如 DamID/LaminB1）对 B compartment 的空间约束变化
- Degeneracy-Breakers:
  - 时间序列 KO：若 TAD 快速消失而 compartment 缓慢增强，支持“去屏蔽 + 重排”路径
  - WAPL/PDS5 扰动对照：调挤出驻留时间而非完全 KO，观察 TAD 与 compartment 的非线性响应
  - 改变 A/B 相互作用强度扫描：若 compartment 增强对 epsilon_AB 敏感，支持 A/B 驱动主导
- In Silico Perturbations:
  - virtual Cohesin KO（k_extrude -> 0）
  - virtual CTCF barrier randomization
  - virtual A/B label shuffle（保持比例，破坏序列相关）
  - virtual weakening of A/B attraction in KO background

## Simulation Implementation
- Recommended paradigm:
  - 3D Langevin Dynamics + 事件驱动 loop-extruder（hybrid polymer simulation）
- Recommended engine:
  - OpenMM + 自定义 extrusion 插件/脚本，或 LAMMPS + 外部挤出调度器
- Fidelity / cost tradeoff:
  - 先做低分辨率（50-100 kb/bead）参数扫描，锁定可行区间后再局部提高到 10-25 kb/bead
  - 先单染色体再全基因组，优先复现实验趋势而非逐位点精确匹配
- Explicit solvent / ion note:
  - 初版不需显式溶剂/离子，采用隐式摩擦与有效相互作用；仅在盐浓度依赖现象为核心问题时升级

## Grounding Notes
- Consensus:
  - Cohesin loss 通常导致 TAD/loop 弱化，这是跨系统较一致现象。
  - A/B compartment 可在缺失挤出后保持甚至增强，提示其与挤出机制部分独立。
- Contested:
  - compartment 增强的主因是“去屏蔽效应”还是“活性噪声降低后的近平衡分离”。
  - 不同细胞类型中增强幅度与时间尺度是否一致。
- Heuristic:
  - 用“最小双机制竞争模型 + 正交验证”比单一路径拟合更稳健，可避免把相关性误写成因果性。
